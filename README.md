# My Portfolio

This is a personal portfolio website built with React, Vite, and Tailwind CSS.

## Features

- Responsive design
- Projects showcase
- Skills section
- Contact form

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment

You can deploy this site using platforms like Vercel, Netlify, or GitHub Pages.
